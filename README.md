## 📰 Descrição

Desafio proposto durante o terceiro curso da primeira formação em Java da Alura

Esse projeto simula um **carrinho de compras**, onde você **pode adicionar um produto (nome e preço) para exibi-lo em seu carrinho** *(contanto que seja possível "realizar a compra")*
